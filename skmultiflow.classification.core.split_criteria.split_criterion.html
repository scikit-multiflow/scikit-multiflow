class skmultiflow.classification.core.split_criteria.split_criterion.SplitCriterion[source]¶

Bases: object

Abstract class for computing splitting criteria with respect to distributions of class values. The split criterion is used as a parameter on decision trees and decision stumps.

This class should not me instantiated, as none of its methods are implemented.

Raises:NotImplementedError: This is an abstract class.
