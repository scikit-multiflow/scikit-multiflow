Bases: skmultiflow.classification.core.attribute_class_observers.attribute_class_observer.AttributeClassObserver

Class for observing the class data distribution for a nominal attribute. This observer monitors the class distribution of a given attribute. Used in naive Bayes and decision trees to monitor data statistics on leaves.
